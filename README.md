# 🌦️ Dynamic Weather Information System

## 📌 Project Overview

The **Dynamic Weather Information System** is a web-based application that allows users to check real-time weather information for any city.
It uses the **OpenWeatherMap API** to fetch live weather data such as temperature, humidity, air quality index (AQI), wind speed, and a 5-day forecast.

The system also provides features like **location-based weather detection, search history, safety alerts, and automatic weather refresh**.

---

## 🚀 Features

* 🌍 Search weather by **city name**
* 📍 **Detect weather using current location**
* 🌡️ Display **temperature, humidity, pressure, wind speed**
* 🌅 Show **sunrise and sunset timings**
* 🌫️ Display **Air Quality Index (AQI)**
* 📅 **5-day weather forecast**
* 🕘 **Recent search history**
* ⚠️ **Weather safety alerts** (heat, cold, pollution)
* 🔄 **Auto refresh every 15 minutes**
* 💾 **Local storage support** for search history


## 🛠️ Technologies Used

* **HTML5**
* **CSS3**
* **JavaScript**
* **OpenWeatherMap API**
* **Browser Geolocation API**
* **Local Storage**

## ⚙️ How It Works

1. User enters a **city name** or clicks **Use My Location**.
2. The application sends a request to the **OpenWeatherMap API**.
3. The API returns weather data in **JSON format**.
4. JavaScript processes the data and dynamically updates the webpage.
5. Search history is saved in **local storage** for quick access.

## 🌐 API Used

OpenWeatherMap API
https://openweathermap.org/api
